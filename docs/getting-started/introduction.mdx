Unwrap is a unified API platform that eliminates the complexity of integrating
multiple point solutions. With a single API key, you can convert files, take
clean website screenshots, scrape real-time data, and more.

## Why Unwrap?

Most developers spend days — sometimes weeks — integrating separate services for
file conversion, screenshot capture, and data scraping. Each service has its own
API format, authentication scheme, billing system, and rate limits.

Unwrap gives you one consistent interface:

- **One API key** for all services
- **One billing model** with pooled credits
- **One SDK** that works the same across every endpoint
- **Enterprise-grade docs** with runnable code examples

## Available Endpoints

| Endpoint | Credits | Description |
|----------|---------|------------|
| `POST /v1/convert/pdf-to-docx` | 5 | Convert PDF files to editable DOCX |
| `POST /v1/convert/image-to-webp` | 2 | Compress and convert images to WebP |
| `POST /v1/screenshot` | 3 | Capture clean website screenshots |
| `GET /v1/fuel/prices` | 2 | Get real-time local fuel prices |
| `GET /v1/account/usage` | 0 | View current billing period usage |
