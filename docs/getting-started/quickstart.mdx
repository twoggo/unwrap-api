Get started with Unwrap in two steps.

## 1. Get Your API Key

Sign up at [unwrap.dev](https://unwrap.dev) and copy your API key from the
dashboard. All new accounts start with **500 free credits** — no credit card
required.

## 2. Make Your First Request

<CodeGroup>
```bash curl
curl -X POST https://api.unwrap.dev/v1/convert/image-to-webp \
  -H "X-API-Key: uw_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/photo.jpg",
    "quality": 80
  }'
```

```javascript Node.js
const response = await fetch("https://api.unwrap.dev/v1/convert/image-to-webp", {
  method: "POST",
  headers: {
    "X-API-Key": "uw_your_api_key_here",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    url: "https://example.com/photo.jpg",
    quality: 80,
  }),
})

const data = await response.json()
console.log(data)
```

```python Python
import requests

response = requests.post(
    "https://api.unwrap.dev/v1/convert/image-to-webp",
    headers={
        "X-API-Key": "uw_your_api_key_here",
        "Content-Type": "application/json",
    },
    json={
        "url": "https://example.com/photo.jpg",
        "quality": 80,
    },
)

print(response.json())
```
</CodeGroup>
