## v1.0.0 — May 30, 2026

### Initial Release

- PDF to DOCX conversion endpoint
- Image to WebP conversion with quality and resize options
- Website screenshot capture (viewport, full-page, multiple formats)
- Real-time fuel price lookups by location
- Tiered pricing with Free, Starter, Pro, and Enterprise plans
- API key authentication with key rotation
- Usage tracking and billing via Stripe
- Rate limiting per plan
- Node.js SDK (v1.0.0)
- Python SDK (v1.0.0)
- Comprehensive documentation on Mintlify
