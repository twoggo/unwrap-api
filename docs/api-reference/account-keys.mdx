---
title: "Account Keys"
description: "Create and manage API keys."
---

Create new API keys programmatically.

**Credits per request: 0 (free)**

## Request

```bash
curl -X POST https://api.unwrap.dev/v1/account/keys \
  -H "X-API-Key: uw_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "label": "production-key"
  }'
```

## Response

```json
{
  "success": true,
  "key": "uw_live_a1b2c3d4e5f6...",
  "prefix": "uw_live_a1b2",
  "message": "Save this key — you won't see it again."
}
```
