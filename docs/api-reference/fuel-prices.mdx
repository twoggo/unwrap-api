---
title: "Fuel Prices"
description: "Get real-time local fuel prices near a specified location."
---

Fetch current fuel prices from stations near a given latitude/longitude.

**Credits per request: 2**

## Request

```bash
curl "https://api.unwrap.dev/v1/fuel/prices?lat=40.7128&lng=-74.0060&radius=5&fuelType=regular" \
  -H "X-API-Key: uw_your_api_key_here"
```

## Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `lat` | float | Yes | — | Latitude (−90 to 90) |
| `lng` | float | Yes | — | Longitude (−180 to 180) |
| `radius` | integer | No | `10` | Search radius in miles (1–50) |
| `fuelType` | string | No | `regular` | `regular`, `premium`, `diesel`, or `all` |
